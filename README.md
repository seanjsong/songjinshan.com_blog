# songjinshan.com_blog

A single-page app utilizing [mdblog](https://github.com/seansoong/mdblog.git) as its backend.

## Installation

```
$ git clone https://github.com/seansoong/songjinshan.com_blog.git
$ cd songjinshan.com_blog
$ npm install
```
